# OSI Model - 7 Layers

| Layer | Name         | Example Protocol |
|-------|-------------|-----------------|
| 7     | Application  | HTTP, FTP, DNS  |
| 6     | Presentation | SSL, JPEG       |
| 5     | Session      | NetBIOS         |
| 4     | Transport    | TCP, UDP        |
| 3     | Network      | IP, ICMP        |
| 2     | Data Link    | Ethernet, MAC   |
| 1     | Physical     | Cables, Hubs    |
